Project Experiences 项目经历

Tech

分布式 KV 存储系统（基于 Raft 学习项目）

开发时间：2024.6 – 2024.9
项目描述：
基于导师提供的 Raft 协议分布式 KV 存储系统源码进行学习实践，目标为理解一致性协议核心机制及高并发读写策略。在 Ubuntu 环境中完成系统部署，并将 RocksDB 接入作为底层持久化引擎，完成读写接口适配。

主要工作：

  • 配置与部署 Raft 系统源码，理解日志复制、选举等核心流程；
  • 接入 RocksDB，完成读写接口对接，并支持通过 CLI 进行交互操作；

个人收获：

  • 掌握分布式一致性协议实现流程；
  • 理解 RocksDB 基本使用方式及其在高并发场景中的部署方法。
    项目总结

分布式消息队列系统（基于 Redis 学习项目）

开发时间：2024.10 – 2025.1
项目描述：
该项目基于导师提供的 Redis 消息队列系统源码进行部署与学习，目标为深入理解消息系统的通信模型、发布/订阅机制以及多线程调度策略。项目在Ubuntu 环境中完成源码部署，并通过压力测试分析了系统性能，测试结果显示系统延迟控制在 700ms 内，平均延迟低于 400ms，适用于高并发场景下的数据传输与异步处理。

主要工作：

  • 部署并配置消息系统源码，复现发布/订阅与生产者-消费者模式；
  • 学习多主题与多分区调度策略，了解系统的负载均衡与顺序保障机制；
  • 使用压力测试工具模拟高并发场景，对系统瓶颈进行定位与初步分析；
  • 阅读源码以理解消息编解码（XML/JSON）与多线程处理流程。

个人收获：

  • 理解消息队列系统的核心组件及其在异步通信中的作用；
  • 掌握部署与调试消息中间件的基本流程；
  • 提升了对系统结构、调度机制与性能调优的认知。
    项目总结
