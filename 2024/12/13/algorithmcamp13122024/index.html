代码随想录算法训练营第四十五天|115.不同的子序列 583. 两个字符串的删除操作 72. 编辑距离

先看思路，之后再码

115.不同的子序列

题目链接: 115.不同的子序列

思路都是相似的。先确定dp数组以及其下标的含义
dp[i][j]的含义是以i-1为结尾的s子序列中出现以j-1为结尾的t的个数为dp[i][j]
解下来确定递推公式。当s[i-1]与t[j-1]相等和不相等的时候，情况不同。相等的时候，dp[i][j]有两个部分，一个部分是用s[i - 1]来匹配。还有一部分是不用它来匹配，因为有的时候重复的字符也可以用来匹配，但是序号不同。因此最后递推公式为dp[i][j] = dp[i - 1][j]。
因为是从上方推导而来的，所以有dp[i][0]和dp[0][j]两种情况初始化。当两个下标都为0的时候，就是0。
遍历的时候要从上到下，从左到右，这样可以保证数组能够根据之前计算出来的数值来进行计算。

583. 两个字符串的删除操作

题目链接: 583. 两个字符串的删除操作

两个字符串可以相互删除。
首先确定下标，dp[i][j]以i-1以及以j-1结尾的两个字符串要达到相等所需要删除的元素的最少次数。
和上题同样要区分不同的情况。当word1[i - 1] 与 word2[j - 1]相同的时候，dp[i][j] = dp[i-1][j-1]。不相同的时候有三种情况，删掉word1或者word2，或者两者同时删除，则递推公式为：dp[i][j] = min({dp[i - 1][j - 1] + 2, dp[i - 1][j] + 1, dp[i][j - 1] + 1})。
遍历顺序仍然需要从上到下，从左到右

72. 编辑距离

题目链接: 72. 编辑距离

dp[i][j]表示两个字符串的最近编辑距离。
确定递推公式同样要分情况。当word1[i-1] != word2[j-1]的时候，就需要编辑。当word1删除一个元素的时候，以下标i-2为结尾的word1与j-1为结尾的word2的最近编辑距离，再加上一个操作，也就是dp[i][j] = dp[i-1][j]+1
删除word2的一个元素就是dp[i][j] = dp[i][j - 1] + 1。添加一个字符串的元素就相当于删除另一个字符串的元素。替换元素的时候取最小的dp[i][j] = min({dp[i - 1][j - 1], dp[i - 1][j], dp[i][j - 1]}) + 1。
当下标分别为0的时候，应该分别初始化为i或者j。
需要从左到右，从上到下遍历。
