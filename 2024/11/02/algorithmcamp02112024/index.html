142.环形链表II

题目链接： 142.环形链表II

首先要先判断链表是否有环，然后找到环状结构的入口。这时需要设置fast和slow两个指针。因为fast每次走两步，slow每次走一步，所以如果有环状结构的话，两者总会重合。然后判断环状结构的入口的方法，如图

当fast和slow相遇的时候，fast已经走了x+y+n(y+z)，slow走过的部分为x+y。因为fast指针是一步走两个节点，slow指针是一步走一个节点，所以得到公式：
(x + y) * 2 = x + y + n(y + z)
化简得到：
x = n(y + z) - y（因为这里需要找到入口，也就是x，因此让x单独放一边。

时间复杂度: O(n)
空间复杂度: O(1)
